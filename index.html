What's here?

This is a demo program i put together in 2008 for a university exam on image generation and synthesis. It uses OpenGL fragment shaders to generate a SIRD (Single-Image Random Dots) stereogram of a moving object on screen [SPOILER: It's a teapot, what were you expecting?]

Why is it here?

Pretty much only to try the GitHub platform, since i don't plan to update the demo soon
